MS/PhD Biostatistics

  • BIOS 7747 - Machine Learning for Biomedical Applications: This course provides a theoretical background of unsupervised and supervised machine learning methods and their application to solver biomedical problems. In addition to understanding methodological details, students learn how to use machine learning methods in Python and improve their coding skills.
  • BIOS 7718 - Introduction to Biomedical Image Analysis: This course provides students with the theoretical and computational skills to tackle a biomedical image analysis problem. The course covers important topics such us histogram analysis, image segmentation, morphology, registration, feautre extraction and quantification, and image-based machine learning.
  • BIOS 6640 - R for Data Science: Statistical programming in R, including data management, subscripting, loops, functions, packages, and graphics. Concepts and methods for reproducible research will be covered, as well as computationally intensive statistical methods. These methods are used to analyze data and present results.
  • BIOS 7719 - Information Visualization: Information visualization studies interactive visualization techniques for analyzing abstract data. This course introduces design, development, and validation approaches with applications in various biological and biomedical domains.
